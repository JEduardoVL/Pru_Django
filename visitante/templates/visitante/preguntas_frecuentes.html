¿Qué es el Sistema Aeneta?

El Sistema Aeneta es un repositorio digital creado por el Instituto Politécnico Nacional para almacenar y facilitar el acceso a trabajos terminales y académicos, incluyendo tesis, proyectos de investigación y artículos, con el fin de apoyar la investigación y el aprendizaje dentro de la comunidad del IPN.

¿Cómo puedo buscar documentos en el Sistema Aeneta?

Para buscar documentos en el Sistema Aeneta, utiliza el motor de búsqueda integrado en la plataforma. Puedes realizar búsquedas por palabras clave, autor, título, área de estudio, entre otros criterios, para encontrar los documentos que necesitas.

¿Es necesario ser parte del IPN para acceder al Sistema Aeneta?

Mientras que el acceso a ciertos documentos puede estar restringido a miembros de la comunidad del IPN, muchas de las obras están disponibles para el público general. Es recomendable revisar las políticas de acceso específicas en el sitio web del Sistema Aeneta.

¿Puedo subir mi trabajo académico al Sistema Aeneta?

Si eres estudiante o profesor del IPN y has completado un trabajo terminal o académico puedes subirlos o contactar con algun administrador para que te ayude.

¿Qué beneficios ofrece el Sistema Aeneta?

El Sistema Aeneta ofrece varios beneficios, como la mejora en la búsqueda y acceso a documentos académicos, acceso facilitado a referencias académicas para proyectos e investigaciones, preservación digital del conocimiento, fomento de la colaboración académica y optimización de recursos.

¿Cómo puedo citar los documentos encontrados en el Sistema Aeneta?

Para citar documentos del Sistema Aeneta, sigue el formato de citación recomendado por tu área de estudio o el que se especifica en las directrices para autores de tu institución. Asegúrate de incluir toda la información relevante, como el autor, título del trabajo, año de publicación, y el enlace al documento en el Sistema Aeneta.

¿Qué hacer si encuentro un error en el documento o el enlace está roto?

Si encuentras un error en un documento o un enlace roto, por favor, reporta el problema a través del formulario de contacto disponible en el sitio web del Sistema Aeneta. Proporciona toda la información necesaria para identificar el documento y describir el problema, lo que ayudará a resolverlo más rápidamente.
